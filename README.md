# sports-embassy

### Dependencies
- python3
- virtualenv
- npm

### Installation
    $ git clone git@github.com:sports-embassy/sports-embassy.git
    $ cd sports-embassy
    $ virtualenv . --python=python3
    $ source bin/activate
    $ pip install -r requirements.txt

### Starting the App
    $ python manage.py runserver
    
### Building CSS
    $ make
